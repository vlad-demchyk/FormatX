# Інтернаціоналізація (i18n)

## Підтримувані мови

| Код | Мова | Файл |
|-----|------|------|
| `uk` | Українська | `src/locales/uk.json` |
| `en` | English | `src/locales/en.json` |
| `it` | Italiano | `src/locales/it.json` |

## Ініціалізація

Використовується `i18next` + `i18next-browser-languagedetector`.

**Порядок визначення мови:**
1. localStorage (збережене налаштування)
2. Мова браузера (navigator.language)

**Fallback:** `uk` (українська)

## Налаштування

```typescript
import i18n from "i18next";
import LanguageDetector from "i18next-browser-languagedetector";

i18n.use(LanguageDetector).init({
  resources: { uk, en, it },
  fallbackLng: "uk",
  interpolation: { escapeValue: false },
  detection: {
    order: ["localStorage", "navigator"],
    caches: ["localStorage"],
  },
});
```

## Використання в React

```typescript
import { useTranslation } from "react-i18next";

function Component() {
  const { t } = useTranslation();
  return <h1>{t("appName")}</h1>;
}
```

## Використання в pure JS

```typescript
import { t } from "../app/i18n";
const label = t("images.title");
```

## Функції API

```typescript
// Ініціалізація (з опціональною локалью)
initI18n(locale?: AppLocale): Promise<void>

// Переклад ключа
t(key: string): string

// Зміна мови
setLocale(locale: AppLocale): Promise<void>

// Підписка на зміну мови
onLanguageChanged(cb: () => void): void
```

## Зміна мови в інтерфейсі

Мова змінюється через сторінку Account (`AccountPage.tsx`):
1. Користувач вибирає мову з <select>
2. Зберігається в `AppSettings.locale`
3. Викликається `i18n.changeLanguage(locale)`
4. Встановлюється `document.documentElement.lang`
5. Сторінка перезавантажується (`window.location.reload()`)

## Інтерполяція

Підтримується через `{{key}}` синтаксис:

```json
{
  "footer": {
    "credit": "built by {{author}}"
  }
}
```

```typescript
t("footer.credit", { author: "vl.demchyk" })
// → "built by vl.demchyk"
```

Також використовується в `showConfirm()` для кастомних діалогів:
```typescript
showConfirm("toast.duplicate", { name: "file.pdf" });
```
